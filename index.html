Why

这不是收藏夹，是交付路线

校友真正需要的不是再收藏一套教程，而是知道怎么开始、做什么项目、怎么把结果展示出来。

01

工具跑通

Claude Code / Codex / VS Code / GitHub，不停留在聊天，而是进入项目。

02

项目跑通

每一节都有可检查交付：代码、README、截图、Demo、复盘。

03

作品集跑通

把项目变成校友、面试官和客户能看懂的展示材料。

Route

4 步学习路线

Step 01
AI 编程工具入门
安装、登录、打开项目、让 AI 读取代码和修改文件。
Step 02
标准工作流
需求拆解、生成代码、跑测试、修 bug、写 README、提交版本。
Step 03
6 个实战项目
从 starter 到 demo，练会提示词、代码审查、错误修复和结果包装。
Step 04
作品集表达
GitHub 仓库、项目说明、演示视频、简历项目描述和面试讲法。
