# David Course Hub

David Studio 的纯静态 AI 编程教程站。当前 v2 提供：

- 首页：Claude Code / Codex 双入口
- Claude Code 中文入门教程
- Codex 中文实战教程
- 本地搜索、深色模式、移动端目录
- Catppuccin 明暗主题、终端动效、阅读进度和响应式文档布局

## 打开方式

直接打开 `index.html`，不需要安装依赖或启动服务。

## 验收

```powershell
python test_site.py
```
